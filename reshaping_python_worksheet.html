Purpose. Use pandas melt() to gather repeated columns into a clean, analysis-ready long-format DataFrame.
Core idea. Wide data is often easier to display. Long data is often easier to analyse and plot.

A Wide versus long

temperatures = pd.DataFrame({
  "country": ["Sweden", "Denmark", "Norway"],
  "1994": [1, 2, 3],
  "1995": [3, 4, 5],
  "1996": [5, 6, 7]
})
2. “In wide data, repeated measurements are often spread across several columns.”
3. “In long data, one observational unit may occupy multiple rows.”

B Use melt()

fuels_long = fuels_wide.melt(
  id_vars=["Entity", "Code"]
)

Hint options: use temperatures.melt(id_vars=["country"]).

C Use informative column names

fuels_long = fuels_wide.melt(
  id_vars=["Entity", "Code"],
  var_name="year",
  value_name="oil_consumption"
)
Hint options: specify both var_name and value_name.

D Clean and sort

fuels_long["year"] = (
  fuels_long["year"]
  .str.replace("y_", "")
  .astype(int)
)
fuels_long = fuels_long.sort_values(
  by=["Entity", "year"]
